Arduino
Spirit Box

Type Physical Computing
Tools Arduino Uno, TEA5757
Medium Hardware, 3D Modeling

A physical-computing take on the classic “spirit box”—a device paranormal investigators use to sweep radio frequencies and listen for fragments of sound in static. This build uses an Arduino Uno and a TEA5757 FM module to scan the band automatically, outputting noise and fleeting audio through a speaker so the box feels alive and unpredictable.

Build

The circuit chains the radio module, audio output, and user controls on the Uno. Firmware handles frequency stepping and scanning logic so the listener isn’t choosing stations manually—the box does the hunting. Interaction is tactile: knobs or buttons to start, stop, or shape how aggressively it sweeps.

Enclosure & Process

Before soldering the final hardware, I modeled the enclosure in 3D so components, ports, and speaker placement would fit in one coherent object. That mock-up pass kept the physical build from becoming a breadboard permanently stuck on a desk—the spirit box was meant to be held, carried, and performed with.
