# Isabel Zlatev — Portfolio

Static portfolio site (HTML, CSS, JS).

## Live site

**https://ilz135127.github.io/IsabelZlatevWeb/**

> **Important:** Do not add a custom domain in GitHub **Settings → Pages** unless you own that domain and set up DNS. GitHub will redirect visitors away from the link above and the site will show “can’t be reached.”

## Deploy on GitHub Pages

1. Push to the `main` branch.
2. **Settings → Pages → Build and deployment** → **Source: GitHub Actions**.
3. Wait for the **Deploy site to GitHub Pages** workflow to finish (Actions tab).
4. Open the URL above.

To use a custom domain later, add DNS at your registrar and a `CNAME` file in the repo root.
